# CRUD_API

## Setup and Running

Clone this repo:

```console
$ git clone https://github.com/katusha-2010/CRUD_API.git
```

Go to downloaded folder:

```console
$ cd CRUD_API
```

Install dependencies:

```console
$ npm install
```

Start server:

- running in development mode:

```console
$ npm run start:dev
```

- running in production mode:

```console
$ npm run start:prod
```

- running with horizontal scaling for application:

```console
$ npm run start:multi
```

## Usage

- Create user

- Read user
- Update user
- Delete user
